# sitio_vibecoding

## Pokémon SPA Game

SPA en JavaScript Vanilla (ES Modules) + Tailwind CSS CDN + Hash Routing.

### Ejecutar localmente

Como es un sitio estático, puedes abrir `index.html` con cualquier servidor estático. Ejemplo con Python:

```bash
python -m http.server 8000
```

Luego visita: `http://localhost:8000`.
